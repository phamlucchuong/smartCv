# Frontend Rules (Next.js / TypeScript)

- Uses the Next.js 14 App Router
- Components live under frontend/src/components/
- API calls go through clients under lib/
- Type definitions are in types/api.ts (auto-generated by openapi-typescript; do not edit by hand)
- Styling uses Tailwind CSS
- Tests: `cd frontend && npm test`
- Lint: `cd frontend && npm run lint`
